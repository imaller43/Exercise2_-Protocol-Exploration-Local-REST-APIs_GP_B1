# 🚀 Node-RED REST API Lab 

## 📌 Introduction

This lab explores how to use Node-RED as a lightweight middleware to build and test a local REST API. It demonstrates how devices and applications communicate over HTTP within a local network.

---

## 🎯 Objective

* Deploy Node-RED using Docker
* Create a simple REST API endpoint
* Send and receive data using HTTP (GET/POST)
* Test API using Postman
* Observe data flow using debugging tools

---

## 🛠️ Software Used

* Docker Desktop
* Node-RED
* Postman
* Web Browser

---

## 💡 Benefits of This Lab

* Understand how REST APIs work in IoT systems
* Learn the difference between GET and POST requests
* Gain hands-on experience with local server deployment
* Visualize and debug real-time data flow
* Build a foundation for IoT and cloud integration
